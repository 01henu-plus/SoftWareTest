## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 启动应用
```bash
python app.py
```
访问: http://127.0.0.1:5000

### 3. 运行测试
```bash
pytest test_login.py -v
```

### 4. 生成报告
```bash
pytest test_login.py --html=test_report.html --self-contained-html
```
