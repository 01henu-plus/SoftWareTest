## 🚀 快速开始

### 方法一：使用Live Server（推荐）

1. **安装Live Server插件**
   - 在VSCode中按 `Ctrl+Shift+X` 打开扩展商店
   - 搜索 "Live Server"
   - 安装 Ritwick Dey 的 Live Server 插件

2. **启动登录页面**
   - 在VSCode中打开 `login.html`
   - 右键点击编辑器
   - 选择 "Open with Live Server"
   - 浏览器将自动打开页面

3. **运行自动化测试**
   - 按 `F12` 打开浏览器开发者工具
   - 切换到 "Console" 标签
   - 复制 `test_login.js` 的全部内容
   - 粘贴到控制台并按回车
   - 运行测试：
     ```javascript
     const testSuite = new LoginTestSuite();
     testSuite.runAllTests();
     ```

### 方法二：直接打开文件

1. **打开登录页面**
   - 在文件资源管理器中找到 `login.html`
   - 双击文件，使用默认浏览器打开

2. **运行自动化测试**
   - 同方法一的步骤3

## 💡 手动测试指南

如果您想进行手动测试，可以尝试以下场景：

### ✅ 正向测试
1. 输入 `admin` / `123456` - 应该登录成功
2. 输入 `testuser` / `password123` - 应该登录成功

### ❌ 负向测试
1. 留空用户名或密码 - 应该显示错误提示
2. 输入过短的用户名（如 `ab`）- 应该提示长度不够
3. 输入过短的密码（如 `12345`）- 应该提示长度不够
4. 输入特殊字符（如 `admin@123`）- 应该提示格式错误
5. 输入错误的凭证 - 应该显示登录失败
